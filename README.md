# Moodly - Back End

This is the back-end repository for **Moodly**, a full-stack web application built with React, Node.js, Express, and MongoDB.



## Front-End Repository
The front-end code and full project details here:  
[Front-End Repository on GitHub](https://github.com/TracywLan/moodly-tracker-frontend.git)



## Overview
The back-end server provides RESTful API routes for managing users, authentication, and data persistence using MongoDB. It communicates with the front-end client built in React.



## Technologies Used
- Node.js  
- Express.js  
- MongoDB / Mongoose  
- JWT, bcrypt, dotenv
